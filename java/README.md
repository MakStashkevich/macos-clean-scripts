# Скрипты для очистки Java

Эта папка содержит скрипты для удаления Java Development Kit (JDK) и связанных компонентов.

## Скрипты:

- [`java-delete-jdk.sh`](java/java-delete-jdk.sh): Полностью удаляет все установленные JDK, плагины Oracle Java и остаточные папки.

## Использование:

Для полного удаления JDK Java запустите скрипт:

```bash
./java-delete-jdk.sh
