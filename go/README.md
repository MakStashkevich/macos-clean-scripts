# Скрипты для очистки Go

Эта папка содержит скрипты для удаления Go SDK и связанных компонентов.

## Скрипты:

- [`go-delete.sh`](go/go-delete.sh): Удаляет Go SDK, проверяя, был ли он установлен через Homebrew или напрямую. В зависимости от метода установки, скрипт выполняет соответствующие команды для удаления.

## Использование:

Для удаления Go SDK запустите скрипт:

```bash
./go-delete.sh
