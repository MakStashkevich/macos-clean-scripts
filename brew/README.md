# Скрипты для управления Homebrew

Эта папка содержит скрипты для управления пакетами и приложениями, установленными через Homebrew.

## Скрипты:

- [`brew-lock-save.sh`](brew/brew-lock-save.sh): Сохраняет список всех установленных Homebrew formulae и casks в файл `brew-lock.json`. Этот файл можно использовать для восстановления или синхронизации установленных пакетов.
- [`brew-lock-apply.sh`](brew/brew-lock-apply.sh): Применяет конфигурацию из `brew-lock.json`. Скрипт удаляет лишние пакеты, устанавливает отсутствующие и обновляет существующие до актуальных версий. Также выполняет очистку кешей Homebrew.

## Использование:

1.  **Сохранить текущее состояние Homebrew:**
    ```bash
    ./brew-lock-save.sh
    ```
    Это создаст или обновит файл `brew-lock.json` в текущей директории.

2.  **Применить сохранённое состояние Homebrew:**
    ```bash
    ./brew-lock-apply.sh
    ```
    Этот скрипт прочитает `brew-lock.json` и приведёт вашу систему Homebrew в соответствие с этим файлом.
