# Скрипты для очистки Flutter

Эта папка содержит скрипты для удаления Flutter SDK и связанных компонентов.

## Скрипты:

- [`flutter-delete.sh`](flutter/flutter-delete.sh): Удаляет Flutter SDK, кеши pub-cache, а также символические ссылки Flutter и Dart из `/usr/local/bin`.

## Использование:

Для удаления Flutter SDK и связанных компонентов запустите скрипт:

```bash
./flutter-delete.sh
