# Скрипты для очистки Node.js / npm

Эта папка содержит скрипты для очистки кешей npm и глобальных `node_modules`.

## Скрипты:

- [`npm-delete.sh`](npm/npm-delete.sh): Очищает кеш npm и удаляет глобальные `node_modules`.

## Использование:

Для очистки npm кеша и глобальных `node_modules` запустите скрипт:

```bash
./npm-delete.sh
