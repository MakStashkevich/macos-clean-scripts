# Скрипты для очистки macOS

Эта папка содержит скрипты для очистки различных кешей, временных файлов и неиспользуемых вспомогательных инструментов в macOS.

## Скрипты:

- [`clean-cache-trash.sh`](macos/clean-cache-trash.sh): Очищает пользовательские кеши, системные временные файлы и корзину.
- [`clean-helper-tools.sh`](macos/clean-helper-tools.sh): Сканирует и удаляет неразрешенные вспомогательные инструменты (`PrivilegedHelperTools`), оставляя только те, которые явно указаны в списке `ALLOWED_HELPERS`.

## Использование:

Для очистки кешей и корзины:

```bash
./clean-cache-trash.sh
```

Для очистки вспомогательных инструментов:

```bash
./clean-helper-tools.sh
```
**Внимание:** Перед использованием [`clean-helper-tools.sh`](macos/clean-helper-tools.sh) убедитесь, что список `ALLOWED_HELPERS` в скрипте содержит все необходимые вам инструменты, чтобы избежать случайного удаления.
