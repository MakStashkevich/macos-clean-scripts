# Скрипты для очистки .NET

Эта папка содержит скрипты для удаления .NET SDK и связанных компонентов.

## Скрипты:

- [`dotnet-delete.sh`](dotnet/dotnet-delete.sh): Удаляет .NET SDK, проверяя, был ли он установлен через Homebrew или напрямую. В зависимости от метода установки, скрипт выполняет соответствующие команды для удаления.

## Использование:

Для удаления .NET SDK запустите скрипт:

```bash
./dotnet-delete.sh
